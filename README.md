# Python Learning Journey

A structured collection of Python programs covering core to advanced concepts.

---

## Topics Covered

### 1. Basics & Syntax
- Comments in Python
- Input / Output statements
- Print formatting with f-strings
- Escape characters

### 2. Data Types
- **Numeric:** `int`, `float`, `complex`
- **Boolean:** `bool`
- **String:** string basics, indexing, slicing, methods
- **None Type**
- **Bytes & Bytearray**
- **Range**
- Immutability concepts
- Type casting: `int`, `float`, `complex`, `bool`, `str`
- Base conversion

### 3. Operators
- Arithmetic operators
- Relational / Equality operators
- Assignment operators
- Logical operators (including non-boolean types)
- Bitwise operators
- Shift operators
- Star (`*`) operator
- Power operator
- Ternary & Nested ternary operators
- `==` vs `is` operator
- Plus / Concat operator
- Operator precedence & function priority
- Special operators

### 4. Control Flow
- `if`, `elif`, `else`
- `match` / `case` statements
- `for` loops
- `while` loops
- `break` and `continue`
- Types of control statements

### 5. Strings
- String basics
- String indexing & slicing
- String methods

### 6. Collection Data Types
| Type | File |
|------|------|
| List | `Lists/` |
| Tuple | `tuples/` |
| Set | `sets/` |
| Dictionary | `disctionary/` |
| FrozenSet | `Python_D/DataTypes/` |

- List methods & list comprehension
- Tuple methods & examples
- Set operations
- Dictionary comprehension

### 7. Functions
- Defining and calling functions
- Function scope
- Inner functions
- Functions returning functions
- Passing functions as arguments
- Lambda functions
- Recursion
- Doc strings
- Modules (creating and importing)

### 8. Object-Oriented Programming (OOP)
- Classes and objects
- Instance variables & methods
- Getter & Setter
- Inheritance
- Operator overriding
- Real-world examples: `Employee`, `Person`

### 9. Decorators
- Basic decorators
- Decorators with arguments
- Property decorators
- Decorator chaining
- Multiple decorator examples

### 10. Generators
- Generator basics (`Generator_Latest/`)
